Ubuntu云服务器搭建内网穿透服务

发表于2022-07-09|更新于2022-12-23|云服务器
|字数总计:492|阅读时长:2分钟|阅读量:

本文是在Ubuntu 18.04云服务器上搭建FRP服务端，在windows上搭建客户端进行内网穿透。

  • FRP的GitHub下载链接：https://github.com/fatedier/frp/releases

根据需要下载对应的FRP压缩包

服务器端

  • 下载对应版本的FRP压缩包
  • 解压FRP压缩包
  • 删除frpc.ini和frpc两个文件
  • 修改配置文件frps.ini

修改内容如下：

  • 设置开机自启动
  • 启动/重启/停止frps
  • 打开浏览器输入ip:7500，输入账号密码，能进入frp后台则服务器端配置成功

客户端

windows

  • 浏览器直接下载对应的FRP压缩包
  • 解压FRP压缩包
  • 删除frps.ini和frps
  • 修改配置文件frpc.ini
  • 命令行运行frpc.exe

直接双击运行frpc.exe会提示如下内容：

This is a command line tool.

You need to open cmd.exe and run it from there.

Linux

  • 下载对应版本的FRP压缩包
  • 解压FRP压缩包
  • 删除frps.ini和frps两个文件
  • 修改配置文件frpc.ini

修改内容如下：

  • 设置开机自启动
  • 启动/重启/停止frps

参考文章：UBUNTU18.04安装FRP的配置说明
