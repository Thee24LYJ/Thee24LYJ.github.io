windows系统常用cmd命令

参考视频教程:

合集·电脑cmd命令

1.ipconfig

  • ipconfig：显示IP地址信息
  • ipconfig /all：显示IP地址详细信息(包括DNS等)
  • ipconfig /all | findstr DNS：筛选关于DNS的信息(findstr用于查找特定字符)
  • ipconfig /release：释放所有接口IP地址(即断开网络无法上网)
  • ipconfig /release "WLAN"：释放WLAN接口的IP地址
  • ipconfig /renew：重新获取IP地址(连接网络正常上网)
  • ipconfig /displaydns：显示DNS解析缓存结果(用于网络DNS故障排查)
  • ipconfig /displaydns | clip：复制DNS解析缓存结果到剪贴板中
  • ipconfig /flushdns：刷新DNS解析缓存

2.nslookup

  • nslookup www.baidu.com：显示解析该域名的DNS服务器地址信息及其对应IP地址
  • nslookup www.baidu.com 114.114.114.114：指定访问该域名的DNS地址
  • nslookup -qt=type www.baidu.com：查询域名指定类型的解析记录

type类型如下：

A地址记录（道接查询默认类型） AAAA地址记录 AFSDB Andrew文件系统数据库服务器记录
ATMA ATM地址记录 CNAME别名记录 HINFO硬件配置记录，包括CPU、操作系统信息
ISDN域名对应的ISDN号码 MB存放指定邮箱的服务器 MG邮件组记录
MINFO邮件组和邮箱的信息记录 MR改名的邮箱记录 MX邮件服务器记录
NS名字服务器记录 PTR反向记录 RP负责人记录
RT路由穿透记录 SRV TCP服务器信息记录 TXT域名对应的文本信息等

3.cls

清除屏幕上显示的信息

4.getmac

  • getmac /v：快速查看mac地址

5.powercfg

  • powercfg /energy：跟踪电脑系统60秒内的使用行为并生成电源效率诊断报告
  • powercfg /batteryreport：生成电池使用报告

6.assoc

显示所有系统文件扩展名的关联信息(打开方式)

7.chkdsk

  • chkdsk /f：检查并修复硬盘
  • chkdsk /r：检查并修复物理扇区

8.sfc

  • sfc /scannow：检查系统文件完整性并尽可能修复损坏文件

9.Dism

  • Dism /online /Cleanup-Image /ScanHealth：扫描映像是否损坏但不做任何修复
  • Dism /online /Cleanup-Image /CheckHealth：检查映像是否损坏及是否能修复
  • Dism /online /Cleanup-Image /RestoreHealth：修复映像(在可以修复的前提下)

10.tasklist

  • tasklist：列出系统中正在运行的进程及其详细信息
  • tasklist | findstr explorer：列出系统中关于explorer进程的详细信息

11.taskkill

  • taskkill /f /pid 12345：终止某个进程

12.netsh

  • netsh wlan show wlanreport：生成关于无线网络的报告
  • netsh interface show interface：显示所有网络适配器相关信息
  • netsh interface ip show address：快速查找电脑IP地址信息
  • netsh interface ip show dnsserver：显示DNS服务器相关信息
  • netsh advfirewall set allprofiles state off/on：关闭/打开windows防火墙

13.ping

  • ping www.baidu.com：ping该域名

  • ping www.baidu.com -t：一直ping该域名

14.tracert

  • tracert www.baidu.com：查询访问某主机所通过的网络路径

15.netstat

  • netstat：显示网络连接信息
  • netstat -af：显示所有网络连接及侦听端口
  • netstat -o：显示拥有的与每个连接关联的进程ID
  • netstat -e -t 5：每5秒钟获取一次以太网统计数据

16.route

  • route /?或route --help或route -h等命令：打印route帮助信息

  • route print：查看电脑网络的路由表信息

  • route ADD 157.0.0.0 MASK 255.0.0.0 157.55.80.1 METRIC 3 IF 2：添加静态路由
  • route DELETE 157.0.0.0：删除静态路由

17.shutdown
